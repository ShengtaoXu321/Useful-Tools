# golang脚本---清理电脑缓存

## 说明
* version: 1.0
* golang-Version：go 1.1.6
* 功能：清除windows电脑temp文件夹中30天不常用的文件
* 参考：参考的掘金的@菜鸟小白的学习分享的Python代码思路，使用golang进行重构
* 优势：跨平台使用，无需担心是否存在环境配置问题。golang支持交叉编译到Windows、linux、macos平台。

## Tips
* 在对比Python和Golang风格的时候，真的喜欢Python这种弱语言的风格，无须担心格式问题
* Golang底包使用结构体封装了，有点像C++的类的概念，可以多瞻仰学习
